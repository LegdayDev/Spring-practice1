# Spring-practice1
스프링 MVC 1강의 : 섹션7 프로젝트

## Dependencies
- Build 도구 : Gradle
- SpringBoot Version : 2.7.14
- Java Version : 11
- Spring Web, Lombok, Thymeleaf
- H2Database(1.4.199)

## 구현기능
- 상품 목록 보기(완료)
- 상품 상세 보기(완료)
- 상품 등록(완료)
- 상품 수정(완료)

- 삭제는 따로 만들어보기(예정)

- 현재는 static 으로 Map을 선언해 내부에서 데이터를 저장하고 관리하지만 H2Database 를 연결하여 JPA를 통해 구현예정
